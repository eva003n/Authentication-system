## Authentication system
This is a guide to learning authentication and authorization for backend engineers by implementing your own authentication system from scratch

***Note:*** **Its not recommended to build your own authentication system in production since its easy to make mistakes and the consequences is security issues which is expensive, use an implemented solution from auth providers**

In this guide we start from the basic form of authentication by standard http and build our way from their to more advances concepts

Security will be a priority while implementing this system

We will also discuss the pros, cons and use cases of each form of authentication

This guide is just the implementation using the following stack

- Backend
    1. Node & express js
    2. Database(Mongo Db)

- Frontend
  Server side rendered webpages
  1. HTML
  2. Css,
  3. Ejs
